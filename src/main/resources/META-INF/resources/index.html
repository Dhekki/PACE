Guia da API REST

Roteirização de veículos com capacidade e janelas de tempo - integração via cURL

1. Baixar dados de demonstração

              
              curl -X GET -H 'Accept:application/json' http://localhost:8080/demo-data/FEIRA_DE_SANTANA -o sample.json

2. Enviar dados para otimização (POST)

A operação POST retorna um jobId que deve ser usado nos comandos subsequentes.

              
              curl -X POST -H 'Content-Type:application/json' http://localhost:8080/route-plans -d@sample.json

3. Obter status e score atuais

              
              curl -X GET -H 'Accept:application/json' http://localhost:8080/route-plans/{jobId}/status

4. Obter o plano de rotas completo

              
              curl -X GET -H 'Accept:application/json' http://localhost:8080/route-plans/{jobId}

5. Encerrar otimização antecipadamente

              
              curl -X DELETE -H 'Accept:application/json' http://localhost:8080/route-plans/{jobId}
